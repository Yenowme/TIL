### 명령어

- SHOW - DB,테이블을 보는 **명령어**.

    `show databases` : 모든 db확인 가능

    `show tables` : 모든 테이블 확인 가능

- CREATE - DB,테이블을 만드는 **명령어**.
- SELECT - 데이터를 보는 **명령어**.
- INSERT - 레코드를 삽입하는 **명령어**.
- UPDATE - 데이터를 업데이트하는 **명령어**.
- DELETE - 레코드를 삭제하는 **명령어**.

---

### 사용

- **테이블 구조 확인**
    1. desc tablename;
    2. describe tablename;
    3. explain tablename;
- **데이터베이스 삭제**
    - drop database dbname
